# Zonaite

一个简短的项目描述。

## 安装

```bash
pip install zonaite
```

## 使用方法

```python
from zonaite import example

# 使用示例
```

## 开发

1. 克隆仓库
```bash
git clone https://github.com/yourusername/zonaite.git
cd zonaite
```

2. 安装开发依赖
```bash
pip install -r requirements.txt
```

3. 运行测试
```bash
python -m pytest
```
